# ATRIX

Painel de acompanhamento de chamados B2B.

## Arquivos principais

- `painel_atrix.html`: painel web para carregar a planilha e acompanhar SLA, MOP/EPS, regiao, substatus e tickets.
- `encerrados.html`: painel executivo dos chamados encerrados, com graficos, projecoes, reincidencia e tabela completa.
- `PAINEL_ATRIX.xlsx`: planilha completa usada como base de acompanhamento.
- `BASE.xlsx`: base simples de chamados.
- `GESTAO_B2B_CHAMADOS_RELATORIO_EXECUTIVO_encerrados.xlsx`: base executiva de chamados encerrados.

## Como abrir

Abra `index.html` ou `painel_atrix.html` no navegador e carregue a planilha XLSX pela tela. Para a analise de encerrados, use o botao "Chamados encerrados" no menu lateral ou abra `encerrados.html`.
